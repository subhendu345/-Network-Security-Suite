# 🚀 Network Security Suite

## Complete Network Automation & Security Toolkit

### 📋 Overview
This suite contains 5 professional network security projects with automation scripts, perfect for DevOps and Security portfolios.

### 🎯 Projects Included
1. **Network Monitor** - Real-time network scanning and monitoring
2. **SSH Honeypot** - Fake SSH server to catch attackers
3. **Firewall Manager** - Advanced firewall automation
4. **IP Intelligence** - IP reputation and threat analysis
5. **Packet Analyzer** - Custom packet sniffer and analyzer
